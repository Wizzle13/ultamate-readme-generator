The Ultamate README Generator Intro

This is not the Ultamate README Generator, this is only a over view of the Ultamate README Generator. This project was designed to help simplify the process of creating Professional and uniformed README file for my projects.

The Instalation of the Ultamate README Generator can be done in just a few easy steps
  1. Go to GitHub Repository
  2. Clone the Repository
  3. In the Command line run : node index.js
  4. Anwser the Questions
  5. Once finished copy the README.md from the dist folder to your project folder
  6. Make any minnor adjustment
